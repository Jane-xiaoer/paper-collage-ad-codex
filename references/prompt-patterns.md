# Prompt Patterns

## Style anchor keyframe

Use case: ads-marketing.

Asset type: finished 16:9 or 9:16 scene keyframe.

Input images: identify the official brand reference, product UI reference and any approved style reference by role.

Scene: describe one visual idea and one physical joke. Name every important paper character and prop.

Style: physical layered editorial paper collage, photographed from directly above, flat even lighting, fine uncoated-paper field, black-and-white round-dot halftone, crisp machine-cut edge, thin warm-white keyline and soft down-right shadow.

Brand: state exact palette values and which elements may use the accent color.

Composition: place the dominant action across at least 40 percent of the frame. Reserve deliberate negative space for the short label.

Text: quote the exact label and support copy. Require verbatim rendering and prohibit extra characters.

Avoid: glossy 3D, neon, purple-blue technology gradients, photorealistic people, childish faces, dirty torn edges, heavy shadows, malformed text, extra logos and perspective distortion.

## Continuity keyframe

Start with: "Image 1 is the strict style reference. Preserve its paper field, halftone density, cut edges, shadows, lighting and character proportions. Image 2 is the previous scene continuity reference. Image 3 is the official brand reference."

Describe only the new scene action after restating the fixed craft system. Reuse recurring characters with the same props.

## Prompt-driven motion

Open on the empty paper field. Introduce each named paper element from a specific edge in story order. Use crisp stop-motion timing and smooth long-tail settling. Settle the soft shadow under each element as it lands. Reveal the short label last. End on the supplied finished keyframe and hold. Use flat even lighting. Prohibit camera wobble, zoom drift, object morphing, extra text and new objects.
